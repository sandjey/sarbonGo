Ролевой flow: DM, Driver, CM (пошаговый сценарий)

Каждая роль отдельной веткой: регистрация → connection offers → cargo offers → один активный trip → этапы → оценки.

Swagger
Водитель Менеджер по грузам (CM) Менеджер по водителям (DM)
Дерево сверху вниз: сначала общий старт, затем отдельные ветки ролей, потом общая точка завершения.
Синий — водитель, оранжевый — менеджер по грузам, зеленый — менеджер по водителям.
